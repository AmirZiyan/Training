    List in HTML

    List is used to group together related item.

    Mainly there are three types of List

    • Ordered List
    • Unordered List
    • Definition List

    1. Ordered List

      Ordered List in HTML to show items in a specific order.Here , Each item is inside an li tag . For Eg:

      1. HTML
      2. CSS
      3. JAVASCRIPT

    2. Unordered List

      It is another kind of List, which doesn't show the items in specific order. Basically shown in bulletin.. For Eg:

      • Mango
      • Orange
      • Chikku

    3. Definition List

      The third a and last of the list is DL, which is used for term-definition pairs, like dictionaries.
